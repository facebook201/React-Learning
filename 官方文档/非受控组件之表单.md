### 非受控组件



非受控组件和受控组件的区别



> 非受控组件 就行传统的HTML表单输入

















结论



| 特征                    | 不受控制 | 受控 |
| ----------------------- | -------- | ---- |
| 一次性价值检索 例如提交 | ✓        | ✓    |
| 在提交时验证            | ✓        | ✓    |
| 即时现场验证            | X        | ✓    |
| 有条件地禁用提交按钮    | X        | ✓    |
| 强制执行输入格式        | X        | ✓    |
| 一个数据的几个输入      | X        | ✓    |
| 动态数据                | X        | ✓    |
